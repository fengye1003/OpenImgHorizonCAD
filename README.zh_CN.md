# ImgHorizon CAD
### 🚀项目概述
幻域 · CAD是一个使用 .NET 开发的 AutoCAD 插件程序库。本插件程序库由来自相互科技工作室（teko.IO SisTemS!）与南京工业大学（NJUT/NJTECH）的 幻愿Recovery （HYrecovery）开发和维护，定位是一组 集成现代功能、助力建筑系学生更好地使用CAD、更容易地学会CAD 的开源CAD插件集。这同时也是建筑类大学生创新创业项目【COArCHitecture】的一部分。
### ✔️核心功能
目前该项目仍在起步和功能添加阶段，目前实现了一个可在AutoCAD中唤出和使用的AI Agent功能，并可以将命令一键发送到CAD控制台，执行各种操作。

### ❤️使用&参与贡献
我们欢迎所有类型的贡献，你可以自由发Issue和PR。

使用文档仍在制作中，不过大部分功能根据程序提示使用即可。你可以使用"netload"命令在CAD中加载此插件，然后通过"hy"命令启动面板。

使用、编辑、Fork、再创作此应用，均需遵循所有已发布的使用协议和开源协议。

部分资源收集自互联网，此项目的核心目的是学习和研究新技术，如有侵权可联系我们的团队，我们将会尽快处理。更多信息请参阅我们的网站。
